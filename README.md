# Напарник — ИИ-ассистент продуктивности

Смотрит на экран раз в пару минут, понимает, чем ты занят (vision-LLM),
ведёт автоматический учёт времени, помогает держать фокус и подводит
итоги дня — с характером доброго друга, а не надзирателя.

## Запуск

```bash
pip install -r requirements.txt

# Веб-панель + наблюдение (одним процессом):
python -m uvicorn app.server:app --port 8010
# → http://localhost:8010

# Или по отдельности, в консоли:
python -m app.watcher            # наблюдение
python -m app.report_cli         # сводка дня + вечерний отчёт
python -m app.focus_cli "дописать курсовую" --minutes 60
```

Ключи API: `~/.groq/key.txt` (основной) и `~/.openrouter/key.txt` (запасной).

## Как это устроено

```
скриншот (mss) ──> vision-LLM (Groq llama-4-scout) ──> JSON {категория, описание, отвлечение}
                        │ отказ? → OpenRouter (nemotron-nano-vl)
                        v
                 SQLite (observations) ──> статистика дня, стрики
                        │
                 Напарник (LLM + личность) ──> реплики, вечерний отчёт, чат
                        │
                 FastAPI + веб-панель (static/index.html)
```

| Модуль | Отвечает за |
|---|---|
| `app/config.py` | все настройки и категории |
| `app/capture.py` | скриншот → маленький JPEG в памяти |
| `app/vision.py` | классификация активности, два провайдера |
| `app/db.py` | SQLite: наблюдения и фокус-сессии |
| `app/reporter.py` | минуты по категориям, доля отвлечений, стрики |
| `app/companion.py` | личность Напарника, отчёты, реплики, чат |
| `app/watcher.py` | цикл наблюдения + реакции |
| `app/server.py` | FastAPI: API + фоновый поток наблюдения |
| `static/index.html` | веб-панель |

## Приватность

Скриншот живёт только в оперативной памяти: сжимается, отправляется
модели и забывается. На диск сохраняется только текст («код — пишет
Python»), без содержимого экрана.
